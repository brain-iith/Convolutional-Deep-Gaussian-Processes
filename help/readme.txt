gpflowrc is a setting file. It help is fixing float type. (very essential for choleskey error).

dgp_train4 and wconv_train4 files are the one needed to be run.
loading_model is an example of loading a pretrained model.
After the file is being run, logs, nlpp, accuracy and time per epochs will be saved in seprate folder. (at step size of 5 epcohs, user can change that in optimizer function.)
data folder contains the data. The example files present are for convex non convex dataset.
